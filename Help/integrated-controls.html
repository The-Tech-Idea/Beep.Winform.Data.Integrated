BeepForms & BeepBlock — Data-Bound Application Framework

BeepForms is the top-level application shell. BeepBlock is the data-bound content panel inside it. Together they provide a zero-code data-entry/CRUD experience driven entirely by BeepBlockDefinition objects.

Architecture

BeepForms (application shell)
  ├── BeepFormsToolbar          ← Save, Refresh, Actions toolbar
  ├── BeepFormsCommandBar       ← Undo/Redo, Cancel, navigation
  ├── BeepFormsQueryShelf       ← Filter/search query builder
  ├── BeepFormsHeader           ← Title, breadcrumb, metadata
  ├── BeepFormsPersistenceShelf ← Savepoint management
  ├── BeepFormsStatusStrip      ← Status messages, progress
  └── BeepBlock (content panel)
        ├── RecordMode            ← Single-record CRUD form
        ├── GridMode              ← Multi-row data grid
        └── QueryMode             ← Query builder + results

Rapid setup

var forms = new BeepForms();
forms.SetDefinition(new BeepBlockDefinition
{
    BlockName = "CustomerForm",
    PresentationMode = BeepBlockPresentationMode.Record,
    Entity = new BeepBlockEntityDefinition
    {
        DataSourceName = "northwind.db",
        EntityName = "Customers"
    },
    Fields = new List<BeepFieldDefinition>
    {
        new() { FieldName = "CustomerId", ControlType = "BeepTextBox" },
        new() { FieldName = "CompanyName", ControlType = "BeepTextBox" },
        new() { FieldName = "ContactTitle", ControlType = "BeepComboBox",
                LOVDataSource = "northwind.db", LOVEntity = "ContactTitles" }
    }
});
forms.Initialize();

ModeDescription

BeepAppTree — Application Navigation Tree

Extends BeepTree with branch-driven navigation from BranchesClasses.

Structure

public IErrorsInfo CreateRootTree()
{
    // Phase 1: Genre branches → category headers
    foreach (var def in BranchesClasses.Where(b => b.BranchType == Genre))
        CreateNode(id, activator());

    // Phase 2: Root branches → matched to parent genre
    foreach (var def in BranchesClasses.Where(b => b.BranchType == Root))
        AddBranch(parentGenre, activator());

    // Phase 3: CreateChildNodes() builds subtrees recursively
}
// Event pipeline: PreShowItem → CreateRootTree → SetConfig → CreateChildNodes

Branch handler

public interface ITreeBranchHandler
{
    IBranch GetBranch(string text);
    IErrorsInfo AddBranch(IBranch parent, IBranch child);
    void RemoveNode(IBranch branch);
    void RefreshTree();
}

Master-Detail binding

var master = new BeepForms();
var detail = new BeepForms();
master.Block.SetDefinition(masterDef);
detail.Block.SetDefinition(detailDef);

master.Block.UnitOfWorkActivity += (s, e) =>
{
    if (e.ActivityType == UnitOfWorkActivityType.CurrentChanged)
        detail.Block.Bind(master.Block.FormsHost, master.Block.BlockName);
};
